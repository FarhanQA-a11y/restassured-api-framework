# Rest Assured API Automation Framework

## Overview

This project is a sample API Automation Framework developed using Java, Rest Assured, TestNG, and Maven.

The framework demonstrates API validation, response verification, status code validation, and JSON response parsing.

## Tech Stack

* Java 17
* Rest Assured
* TestNG
* Maven
* Git
* GitHub

## Project Structure

src/test/java

* tests

  * UserApiTest.java

## Features

* API Automation Testing
* GET Request Validation
* Status Code Verification
* Response Body Validation
* JSON Parsing
* TestNG Integration
* Maven Build Execution
